## 📌 About The Project

TwinText is a NLP-based platform designed to analyze textual content for originality, authenticity, and quality. The system combines Machine Learning, Natural Language Processing, and modern web technologies to provide a complete content analysis solution.

The platform supports:

- 🔍 Plagiarism Detection
- 📝 Paraphrase Detection
- 🤖 AI Content Detection
- 🌐 Language Translation
- ✨ Grammar Checking
- 📄 PDF Upload & Text Extraction

TwinText helps students, researchers, educators, writers, and professionals ensure content integrity and improve writing quality.

---

## ✨ Key Features

### 🔍 Plagiarism Detection
- Detects copied and semantically similar content
- Supports text and PDF inputs
- Generates similarity percentage
- Uses SBERT embeddings and Cosine Similarity

### 📝 Paraphrase Detection
- Identifies whether two texts convey the same meaning
- Semantic similarity analysis using NLP models
- High-accuracy paraphrase classification

### 🤖 AI Content Detection
- Classifies content as AI-generated or Human-written
- Machine Learning based prediction model
- Probability-based output

### 🌐 Language Translation
- Multi-language text translation
- Powered by Google Translate API

### ✨ Grammar Checker
- Detects grammar and spelling mistakes
- Suggests corrections
- Powered by LanguageTool API

### 📄 PDF Processing
- Upload PDF documents
- Automatic text extraction
- Analysis of extracted content
